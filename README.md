# Portfolio personnel – Exploration d’agents IA

## Description
Ce projet a deux objectifs principaux :

- Étudier les forces, les limites et les compromis des agents d’intelligence artificielle dans un contexte concret.
- Servir de vitrine personnelle sur le Web afin de présenter mon profil, mes intérêts techniques et mes expérimentations.

Le projet est conçu comme un terrain d’expérimentation évolutif, autant sur le plan technique que conceptuel.

## Objectifs
- Comprendre le comportement et les capacités des agents IA.
- Tester différentes approches d’intégration et d’orchestration d’agents.
- Appliquer de bonnes pratiques frontend, backend et SEO.
- Créer une présence Web simple, performante et maintenable.

## Technologies utilisées
- **Next.js** : rendu côté serveur (SSR), performance et SEO.
- **Node.js** : logique backend et intégrations.
- **SEO (Google)** : optimisation du référencement via mots-clés, métadonnées et structure de page.
- **Github Copilot** : Les principaux agents utilisés selon le contexte : GPT5-Mini, Claude Sonnet 4.5, Claude Opus 4.5

## Hébergement
- **Vercel (offre gratuite)** : déploiement continu, performance globale et simplicité d’infrastructure.

- Projet personnel en évolution.

Ce projet n’a pas vocation commerciale. Il sert principalement à l’apprentissage, à l’expérimentation et à la démonstration de compétences techniques.
